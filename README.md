# FileHoster API Wrapper

Its a simple API Wrapper for Python


## Supported / Unsupported Hoster

- [ ] Mediafire
- [x] Anonfile
- [ ] Mega.nz
- [ ] Upload.me
- [x] Vidoza
- [x] Vivo


## Usage

```python
import main as FileHoster

vivo = FileHoster.Vivo(vivotoken)
accinfo = vivo.AccInfo()
print(accinfo)
```
